# Thyblog - CMS client

Single page application (SPA) front-end for writers of Thyblog site. Public front-end built with React, Vite and React Router v6.

**Links**:

- [Server](https://github.com/MewKay/blog-api-server)
- [Public](https://github.com/MewKay/blog-api-client-user)

## Features

- List and browse authors posts.
- Render posts from Markdown using react-markdown.
- Authenticated authors can fully manage posts they created as well as comments under it.
- Rich post editing with MDXEditor.
- Custom form validator.
- Components tested with React Testing Library and Vitest.
